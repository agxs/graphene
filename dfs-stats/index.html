Digimap Logfile Stats

  • Maps/Minute graphs count the number of requested maps in a minute.
  • Response times are all in milliseconds.
  • Max Response is the maximum response time from the aggregate of the average map times for each map type.
  • Avg Response is the average response time to generate all the maps in that minute.
  • Backends is the number of database users connected.
  • Xacts is the number of transactions(not queries).
  • Blocks is the number of blocked transactions. Anything non-zero is very bad!
